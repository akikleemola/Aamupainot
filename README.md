# Aamupainot

## Sovelluksen toiminnot

* Käyttäjä pystyy luomaan tunnuksen ja kirjautumaan sisään sovellukseen.

* Käyttäjä pystyy lisäämään päivän aamupainon päivämäärän ja painon perusteella.

* Käyttäjä näkee listan omista aiemmista painomerkinnöistään.

* Käyttäjä pystyy muokkaamaan omia painomerkintöjään, jos päivämäärä tai paino on kirjattu väärin.

* Käyttäjä pystyy poistamaan omia painomerkintöjään.

* Käyttäjä näkee painon kehityksen graafisena kaaviona.

* Käyttäjä näkee yhteenvetona viimeisimmän painon, kokonaismuutoksen ja tallennettujen merkintöjen määrän.

* Sovellus näyttää vain kirjautuneen käyttäjän omat painomerkinnät.

* Sovelluksessa on tietoturvaa parantavia toimintoja, kuten salasanan hajautus, CSRF-suojaus, käyttäjäkohtaiset tietokantakyselyt ja lomaketietojen validointi.

* Sovelluksessa on tumma dashboard-tyylinen käyttöliittymä.
